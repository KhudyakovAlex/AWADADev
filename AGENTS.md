# AGENTS.md

## Чем занимаемся вообще

Цель репозитория / этой работы с агентом: **правильно организовать процесс разработки ПО** в компании (автоматизация инженерных систем зданий, сооружений, территорий).

Конечный интерес:
- целостность разработки между отделами;
- общий стиль интерфейса;
- возможность перераспределять ресурсы между проектами/задачами.

Разработка ПО в фирме — **не главное** направление бизнеса, но критичное звено. Основная деятельность фирмы — **реализация проектов автоматизации** на базе продукта **СУ**.
