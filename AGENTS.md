# AGENTS.md

## Чем занимаемся вообще

Цель: **организовать процесс разработки ПО** в компании (автоматизация инженерных систем зданий / сооружений / территорий).

Конечный интерес позже: целостность разработки, единый UI-стиль, перераспределение ресурсов между проектами.

Сейчас разработка ПО — не главное направление бизнеса, но критичное звено. Бизнес = **проекты автоматизации** на базе **СУ**.

---

## Текущий этап

Построить **чёткую модель компании**: сущности, типы, связи, дыры.

Ещё не делаем: целевой процесс, регламенты, план внедрения.

### Канонические артефакты

- `docs/struktura-kompanii.md` — первичная база модели компании: сущности, реестр, связи, дыры, ID.
- `docs/opisanie-kompanii-dlya-vneshnego-chitatelya.md` — обязательное полное и достаточно формальное описание компании для человека без знания внутренних ID; пользовательский интерфейс к базе.

База первична. При изменении фактов сначала обновляется `docs/struktura-kompanii.md`, затем синхронно обновляется `docs/opisanie-kompanii-dlya-vneshnego-chitatelya.md`.

### Идентификаторы

- Кириллица, **ПРОПИСНЫЕ**, разделитель `-`, иерархия в пути.
- Отделы **плоские**: `О-П`, `О-И`, `О-РП`, `О-РПИ`, `О-РПБИМ`, `О-РПБИС`, `О-РУ`.
- Должности: `Д-ПРОГ`, `Д-ТЕСТ`, `Д-ПРОЕКТ`, `Д-ДИЗ`, `Д-ИНЖА`, `Д-КОНСТ`, `Д-СХЕМ`.
- Сотрудники: `С-АХ`, `С-ЮБ`, `С-АШ`, `С-БК`, `С-ЕК`, `С-ИШ`, `С-ДС`, `С-АЧ`, `С-ВБ`, `С-ВК`, `С-АП`, `С-ОГ`, `С-ДБ`, `С-ЕД`, `С-АС`; заглушки `С-ОИ-*`, `С-ОП-*`, `С-РУ-*`.
- Изделия СУ: `СУ-А-Р`, модули `СУ-А-М*`, `СУ-А-ДПО`, `СУ-А-НПКС`, `СУ-А-СИН`…
- Инструменты: `ПИ`, `ПИ-ТС`. ПО объекта: `СУ-П-БЭК`, `СУ-П-ФРОНТ`, `СУ-П-Д`.
- Продуктовая ветка: `СУ-А-СИН` (О-РУ) + `СУ-П-СИН` (О-РПБИС) — отдельно от проектного контура.

### Правила для агента

1. Факты и структура первично согласуются в `docs/struktura-kompanii.md`.
2. Не выдумывать; неизвестное — §5.
3. ID по правилам выше; старые `оП` / `СУ-ПИ-TS` не использовать.
4. Внешнее описание компании держать отдельно в `docs/opisanie-kompanii-dlya-vneshnego-chitatelya.md`: без ID, но полно и формально.
5. Отвечать кратко.
